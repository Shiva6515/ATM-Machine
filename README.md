\# 🏧 ATM Machine — Java Console Application



\## 📌 About

A simple ATM machine simulation built using Java.

It allows users to log in with a PIN and perform basic banking operations.



\## ✨ Features

\- 🔐 PIN verification with 3 attempts

\- 💰 Check account balance

\- ➕ Deposit money

\- ➖ Withdraw money

\- 🚪 Exit option



\## 🛠️ Tech Stack

\- Language : Java

\- Concepts Used : OOP, Static Methods, Scanner, Loops, Switch-Case



\## 🚀 How to Run



1\. Make sure Java is installed on your system

2\. Clone this repository :

&#x20;  git clone https://github.com/Shiva6515/ATM-Machine.git

3\. Open the project folder

4\. Compile the file :

&#x20;  javac atmMachine.java

5\. Run the program :

&#x20;  java atmMachine
